# MediaPipe Hands Experiment

This repository contains a small experiment using **[MediaPipe Hands](https://google.github.io/mediapipe/solutions/hands)**, a library for real-time hand tracking.

## Overview

- The code captures video from the webcam.
- MediaPipe Hands detects hand landmarks in real-time.
- Landmarks are drawn onto a `<canvas>` overlay.
- Basic gesture detection logic is applied (e.g., checking if fingers are extended).

## Key Features

- Uses **MediaPipe Hands** for hand tracking.
- Works directly in the browser with **JavaScript**.
- Draws detected hand landmarks on a canvas for visualization.
- Demonstrates a simple approach to detecting gestures based on landmark positions.

## Notes

- This is a playground to understand how MediaPipe Hands works.
- The focus is on learning the library, not on building a full application.
- Gesture detection logic is basic and for experimentation purposes.
